# Capstone_Braille
졸업작품 프로젝트: 스마트폰 연동 점자정보단말기
<br> 
<br> 
<br> 

## 동작
1. 스마트폰에서 앱 실행
2. 카메라 사용 - 텍스트 인식(ML Kit Text-recognition 적용)
3. 인식한 텍스트 라즈베리파이로 BLE 전송
4. 인식한 텍스트 TTS 출력
5. 라즈베리파이에서 BLE 사용해 스마트폰과 연결
6. 텍스트 수신
7. 시리얼로 아두이노 전송
8. 아두이노에서 한글 자모음 분리 및 점자 모듈 출력
<br> 
<br> 
<br> 
## HW
1. 스마트폰
